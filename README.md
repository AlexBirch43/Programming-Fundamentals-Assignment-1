# Wyre Systems Group: Component tracking and logging tool

## Setup
- Ensure Python and Pycharm are installed on your PC, and are updated to the latest versions. No additional libraries are required as this only uses built in modules.
- Ensure Microsoft Office is also installed on the PC and is up-to-date.
- Ensure the latest versions of all files from Component_Tracker from this repository is downloaded to your PC.
- Download the CSV standard template from this repository into the same location as the .py files. (This is not essential as the program can create a new CSV if required)
- Open main.py and run the program.

  
## Basic Functions - Please see Figure 2 within the release documentation for further information.
- The program will create a new CSV if one does not exist.
- Login system for role based access.
- Role-based menu pages.
- Search function.
- Use item.
- Check stock Remaining.
- Edit Items (add remove stock).
- Add New Item
- Remove Item
- Log all actions where stock levels are changed


## Using the Inventory System
- Login depending on your user role.
   - Credentials for Engineer are      : Username Engineer   Password ENGINEER
   - Credentials for Manager are       : Username Manager    Password MANAGER
- Once logged in, the relevant menu will be displayed.
- From here you can navigate through the system by selecting the required function using the relevant numerical key and ENTER.
- Each function will contain prompts to explain it's use.
- Use the Logout option in the menu when you are finished.
